# Mini CRM â€” Client & Lead Management System

A full-stack Mini CRM web application that captures leads from a contact form and manages them through an admin dashboard. Built using Node.js, Express, MongoDB, and vanilla frontend technologies, this project demonstrates real-world CRM workflow functionality and backend API integration.

---

# Features

# Lead Capture
- Contact form to collect user details
- Automatically saves leads to database
- Default lead status set to `new`

# Admin Dashboard
- View all leads in table format
- Update lead status (new / contacted / converted)
- View submitted notes
- Simple dashboard layout with sidebar navigation

# Backend API
- Create new leads
- Fetch all leads
- Update lead status
- Update notes
- Delete leads
